# Kamera-Hinweise

## Empfohlen
- pro Kamera Mainstream-RTSP eintragen
- für Dashboard optional Substream nehmen, wenn CPU knapp ist
- feste IPs für Kameras vergeben
- Kamera-Zeit/NTP sauber setzen

Die echte Zuordnung machst du danach in der YAML.
